# Lateral Juggler - 次世代ブラウザスロットエンジン

ブラウザ上で動作する高精度なパチスロシミュレーションエンジン。実機の挙動を数学的・視覚的に忠実に再現。

## 1. プロジェクト概要
- **技術スタック**: HTML5 / TypeScript / CSS3 / Canvas API
- **目的**: パチスロ特有の「物理的なリール挙動」と「内部抽選」をWeb標準技術のみで完全再現。

## 2. 特出すべき実装手法

### ① リール制御 (滑り計算アルゴリズム)
- **動的スリップロジック**: レバーON時の内部抽選フラグに基づき、停止位置から最大4コマ（JAC中は21コマ）を瞬時に計算。
- **5ライン完全対応**: 横3線・斜め2線の全ラインを並行してシミュレートし、最適な引き込み（スリップ）位置を動的に決定。

### ② 成立役の評価システム
- **完全排他抽選**: 設定1〜6およびデバッグモードに基づく確率テーブルによる毎ゲームの抽選。
- **多角的判定**: 全リール停止後、5つのライン上のシンボルマトリックスを走査し、ボーナス・小役・リプレイを瞬時に判定・加算。

### ③ 視覚的表現 (3D・ライティング)
- **3Dパースペクティブ表示**: Canvas APIの `drawImage` と座標計算により、リールの円筒状の曲がりと奥行きを再現。
- **バックライト演出**: 当選時のシンボル背後での動的なフラッシュ点滅（CSS Animation / JSタイマー）。
- **GOGO!ランプ告知**: 内部当選状態を監視し、第3リール停止時に告知ランプを点灯。

### ④ データカウンター & スランプグラフ
- **リアルタイム解析**: 全ゲームの差枚数・履歴・ボーナス回数をGameStateで一元管理。
- **動的グラフ描写**: 直近300ゲームの推移をCanvasにリアルタイム描画するスランプグラフ。
